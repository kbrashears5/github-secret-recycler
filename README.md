# Usage
1. Fill in the parameters under the #parameters section in the `secret-recycler.py`
2. Windows Subsystem for Linux
```bash
cd /mnt/c/repos/secret-recycler
```
3. Run Python
```bash
python3 secret-recycler.py
```
